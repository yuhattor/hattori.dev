<p>GitHub Desktop 3.1では、複数のコミット間の差分を表示することができるようになりました。これにより、プッシュしようとしているコミット群の中に意図しない変更がないことを確認することができます。GitHub Desktop 3.2 では、この機能をさらに進化させ、Pull Request をプレビューできるようになりました。機能ブランチからリポジトリのデフォルトブランチに導入されるすべての変更の diff を見ることができます。</p>
